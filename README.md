# CAND_FullProject

Dự án web sử dụng React.js cho Frontend và Node.js cho Backend.

## Cấu trúc dự án

```
CAND_FullProject/
├── client/                 # Frontend React.js
│   ├── public/            # Static files
│   ├── src/               # Source code
│   ├── package.json       # Frontend dependencies
│   └── README.md          # Frontend documentation
│
├── server/                # Backend Node.js
│   ├── src/              # Source code
│   ├── package.json      # Backend dependencies
│   └── README.md         # Backend documentation
│
└── README.md             # Project documentation
```

## Cài đặt

### Frontend
```bash
cd client
npm install
npm start
```

### Backend
```bash
cd server
npm install
npm start
```

## Công nghệ sử dụng

### Frontend
- React.js
- React Router
- Axios
- Material-UI

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
